ElderMeds : Medicine Donation & Request for Old Age Homes

A digital platform to connect medicine donors with old age homes through a managed workflow.

Java JSP / Servlet MySQL HTML ¬∑ CSS ¬∑ Bootstrap

ElderMeds is a web-based platform that streamlines the donation and distribution of valid, unused medicines to old age homes. The system supports three roles: Admin, Donor, and Caretaker.

Donors can register and donate medicines, while caretakers request required medicines for residents. Admins manage users, monitor donation and request logs, and oversee the centralized medicine inventory.

  • Role-based access for Admin, Donor, and Caretaker.
  • Centralized database for tracking donations, inventory, and requests.
  • Reduces medicine wastage and improves timely access for elderly residents.
  • Builds a compassionate, resource-sharing community between donors and old age homes.
